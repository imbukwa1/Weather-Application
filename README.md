Overview

A simple Android Weather Application built using Java in Android Studio. The app fetches real-time weather data from an external API and displays current conditions for a user-selected city.

Features

Search weather by city

Displays temperature, weather condition, and humidity

Clean and user-friendly interface

Tech Stack

Java

Android Studio

OpenWeatherMap API

JSON

XML UI layouts

How It Works

The app sends an HTTP request to the OpenWeatherMap API, parses the JSON response, and displays the weather details on the screen.
